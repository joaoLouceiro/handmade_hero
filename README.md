# handmade_hero

The core of the project is, of course, Casey Muratori's Handmade Hero series. If you somehow stumbled upon this and don't know what I'm talking about, look it up. It's worth it.

The first issue, though, is that the series is based on a Windows system, and, come on.
Fear not, there's this side-project that can help out.

# Handmade Penguin
As per [the project's description](https://davidgow.net/handmadepenguin/default.html):
> Handmade Penguin is a companion to Casey Muratori's amazing Handmade Hero series. As the name suggests, Handmade Penguin chronicles how to follow along if you're using the Linux operating system.
